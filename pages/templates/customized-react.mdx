# Customized React App

The template provides a basic React application with Tailwind CSS, TypeScript and React Router.
You can fork the template and customize it according to your needs by clicking the button below:

[Customized React App](https://github.com/Bilguun1324/my-docs-modules/tree/main/templates/customized-react)

Followings are files and folder structure of the template:
    
- src
  - components
  - pages
    - Home.tsx
    - Auth.tsx
    - Error.tsx
  - styles
    - global.css
  - index.tsx
